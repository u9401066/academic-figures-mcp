# Project Brief

## Purpose

Academic Figures MCP exists to provide an MCP-native workflow harness for planning, generating, evaluating, and iterating publication-ready scientific visuals.

## Target Users

Primary users are AI agents, Copilot-driven workflows, and non-engineering researchers or clinicians who need structured figure-generation workflows without manually stitching together prompts, journal standards, and revision loops.

## Project Summary

Academic Figures MCP is a complete agent harness for publication-ready figure generation workflows. Its primary value is end-to-end orchestration exposed through MCP and packaged as a VSX experience that lets non-engineers adopt it quickly; direct image API integration is secondary infrastructure.

## Requested Capability Expansion (2026-04-11)

- Generate posters as a first-class output type.
- Generate grouped figures and structured multi-panel outputs.
- Generate composite montage figures, such as combining four images into one larger figure.
- Retarget one existing image to different journal requirements while preserving the underlying visual concept.
- Retain prompts and prompt packs so previous outputs can be reproduced.
- Convert image elements or image style cues back into a reusable style prompt for later regeneration.

## Goals

- Emphasize workflow orchestration over raw API connectivity.
- Make MCP + VSX the default adoption path for non-engineering users.
- Show planning, generation, evaluation, and iteration as one integrated agent flow.
- Expand the harness from single-asset generation toward poster, multi-panel, and composite outputs.
- Treat journal retargeting, prompt provenance, and style reuse as product requirements rather than ad hoc operator knowledge.
