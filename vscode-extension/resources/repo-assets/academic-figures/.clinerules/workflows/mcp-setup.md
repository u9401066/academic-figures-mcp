# MCP Setup: Asset-Aware MCP (Local STDIO)

This workflow installs or updates the Academic Figures MCP server entry for Cline.

Use the VS Code extension command so Cline-specific metadata such as
`alwaysAllow` and unrelated servers are preserved.

1. Open this repository in VS Code.
2. Run `Academic Figures: Insert Cline MCP Settings` from the Command Palette.
3. Restart Cline or reload the VS Code window.

Confirm the `academic-figures` server is enabled in the MCP Servers UI.
